# Personal Profile Page

Une page de profil personnel développée en **HTML5 sémantique** dans le cadre de la **Phase 2 - HTML, CSS & Git** de la formation **Akieni Academy**.

Ce projet présente mon profil professionnel à travers une interface simple, accessible et bien structurée.

---

## Présentation

Cette page met en avant mon parcours, mes compétences techniques, ma formation ainsi qu'un formulaire de contact.

L'objectif est de mettre en pratique les bases du HTML5 sémantique tout en respectant les bonnes pratiques d'accessibilité et de structuration d'une page Web.

---

## Objectifs pédagogiques

- Structurer une page Web avec les balises HTML5.
- Utiliser une hiérarchie correcte des titres.
- Organiser le contenu avec des listes.
- Intégrer une image avec un texte alternatif.
- Créer un formulaire accessible.
- Ajouter des liens vers les réseaux sociaux.
- Respecter les standards du W3C.

## Structure HTML5

Le projet utilise les principales balises sémantiques de HTML5 :

- `<header>`
- `<main>`
- `<section>`
- `<footer>`
- `<form>`
- `<dl>`
- `<ul>`
- `<figure>` _(prévu pour une évolution future)_
- Titres (`h1`, `h2`, `h3`)

---

## Accessibilité

Cette page applique plusieurs bonnes pratiques d'accessibilité :

- Texte alternatif sur l'image (`alt`)
- Labels associés aux champs du formulaire
- Attribut `required` pour les champs obligatoires
- Attributs `aria-label` sur les icônes des réseaux sociaux
- Structure logique des titres
- Utilisation des balises HTML5 sémantiques
